3D imagery is the next generation of photography. This project is about using 2D images to make 3D models with dense points, poisson, and texture. A Raspberry Pi with a camera module is used to take 2D images. The camera is triggered from a Laptop controlled by Python program. The images are FTPed (File Transfer Protocol) to the Laptop where they are feature detected using a computer vision algorithm. This makes the dense points by running reconstruction. The MeshLab program makes the poisson and texture by processing the mesh from the 3D dense points model. This is fed into Microsoft Blender which cleans and creates the 3D models in different formats. These formats can be embedded into 3D viewer applications and HTML5 webpages. This is perfect for buisnesses who want to advertise their product on the internet.

Currently, in our schools, attendance is taken by teachers which can be cumbersome and can be prone to errors. This also can take up valuable time of learning, teaching, and uploading the attendance in the system. The engineering objective of this project is to have a well-working, low-cost Artificial Intelligence-based program that can take the attendance of the desired class with 100% accuracy in real-time and be able to generate a report on students attending the class. The Raspberry Pi was set up and the AI-based program OpenCV was installed and compiled. Two python scripts were written first to profile people and the second to recognize them for attendance. I experimented with my program with different people by creating their profiles and running the second script for facial recognition. The program was recognizing people with 100% accuracy. The python script compiled a list of the recognized people and generated an organized attendance report. This whole project was cost-effective and efficient enough to be used in classrooms.

We have so many security devices in our houses, but most of the times when we leave, we forget to arm the devices, creating a security risk. As technology is advancing so are cyber criminals who infiltrate through people's home network. I have successfully created a Python program where it was able to arm and disarm the security devices and the internet, creating an effective and comprehensive automated security arming system. My project involves 3 modules; triggering, security actions, and notifications. When I leave the house, it automatically detects that I am out of the Bluetooth range and as a result, it shuts down the internet, arms Arlo, starts the motion detection system, and sends me a notification. When I enter back into the house (come back in range), it automatically starts back the internet, disarms Arlo, stops the motion detection system, and sends a welcome message to me. This program can be really effective for individuals who want to automatically increase their home security and who want to automate their IoT devices as they exit and enter their houses. This is also effective for businesses and research labs who want to secure their premises and their data.
